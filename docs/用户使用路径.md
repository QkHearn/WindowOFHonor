## 0. 全局说明

### 0.1 路由规划（实现参考）

| 路径 | 页面 | 需登录 |
|------|------|:------:|
| `/login` | 登录页 | 否 |
| `/broadcast` | 荣誉展播（大屏） | 否 |
| `/leaderboard` | 激励排行榜 | 否（「我的名次」需登录） |
| `/supervisor` | 主管工作台首页 | 是（主管） |
| `/supervisor/issue` | 发放激励 | 是（主管） |
| `/supervisor/records` | 发放记录 | 是（主管） |
| `/supervisor/tasks` | 任务令管理 | 是（主管） |
| `/me` | 个人中心首页 | 是 |
| `/me/honors` | 我的荣誉 | 是 |
| `/me/partners` | 最佳拍档 / 关系网络 | 是 |
| `/me/tasks` | 我的任务令 | 是 |
| `/team` | 团队展播 | 是 |
| `/admin` | 系统管理 | 是（超管） |

### 0.2 登录后跳转规则

| 角色 | 默认落地页 |
|------|-----------|
| 主管 / 超管 | `/supervisor` |
| 员工 | `/me` |

### 0.3 视觉基调

全站遵循奢侈品风 UI：墨夜黑 + 象牙白 + 香槟金；登录/大屏深色，操作端浅色编辑式排版。

## 1. 公共路径（无需登录）

---

**角色：** 全员 / 访客  

**目的：** 进入系统或在大屏上展示团队荣誉氛围  

**使用路径：**

1. 打开系统首页或 `/login`
2. 选择「进入」登录，或点击「荣誉展播」进入 `/broadcast`（无需登录）
3. 展播页自动轮播近期荣誉；按 `F11` 或点击「全屏」进入大屏模式
4. （可选）点击导航「排行榜」进入 `/leaderboard` 浏览个人版 / 团队版榜单

---

**角色：** 全员  

**目的：** 使用账号进入系统  

**使用路径：**

1. 访问 `/login`
2. 输入用户名、密码
3. 点击「进入」
4. 系统校验身份 → 按角色跳转：主管至 `/supervisor`，员工至 `/me`
5. 登录失败：底部金色细线提示「凭证无效」，停留登录页

---

## 2. 主管（管理员）

---

**角色：** 主管  

**目的：** 快速掌握本团队荣誉概况  

**使用路径：**

1. 登录 → 进入 `/supervisor`（主管工作台）
2. 查看顶栏：团队名称、本月新增荣誉、活跃成员数
3. 浏览「近期团队动态」时间轴（最近发放 / 获激励事件）
4. 点击「个人排行榜」或「团队排行榜」快捷入口 → `/leaderboard`
5. （可选）点击某成员卡片 → 查看该成员公开荣誉摘要

---

**角色：** 主管  

**目的：** 向一名或多名员工发放激励，形成共获荣誉记录  

**使用路径：**

1. 登录 → `/supervisor`
2. 点击「呈递荣誉」或侧栏「发放激励」→ `/supervisor/issue`
3. **左侧**：在贵宾名册中搜索 / 按团队筛选员工，勾选一名或多名接收人
4. **右侧**：填写激励标题（必填）、说明（选填）、荣誉分值
5. 点击「预览荣誉」→ 弹出证书式确认页（接收人列表 + 激励内容）
6. 点击「确认呈递」→ 提交成功，播放 shimmer 揭晓动效
7. 自动跳转「发放记录」或提示「已写入荣誉展播」
8. 员工荣誉积分即时更新；多人共获时更新拍档关系

---

**角色：** 主管  

**目的：** 复盘历史激励发放情况  

**使用路径：**

1. 登录 → `/supervisor`
2. 点击侧栏「发放记录」→ `/supervisor/records`
3. 浏览时间轴式记录：时间、接收人、标题、分值、操作人
4. （可选）按时间范围 / 接收人筛选
5. （二期）误发时在限时内点击「撤销」并留痕

---

**角色：** 主管  

**目的：** 给员工分配任务令，后续与荣誉挂钩  

**使用路径：**

1. 登录 → `/supervisor`
2. 点击侧栏「任务令管理」→ `/supervisor/tasks`
3. 点击「新建任务令」
4. 选择执行人、填写标题与描述、设置截止时间
5. （可选）关联完成后荣誉分值
6. 提交 → 员工在「我的任务令」中可见
7. （二期）员工提交完成 → 主管确认 → 自动或手动触发激励

---

**角色：** 主管  

**目的：** 查看排行榜，辅助激励决策  

**使用路径：**

1. 登录 → 点击导航「排行榜」→ `/leaderboard`
2. 切换 Tab：**个人版** / **团队版**
3. 切换范围：本团队 / 全公司；周期：本周 / 本月 / 本季 / 累计
4. 查看 Top N 及团队间对标数据
5. 点击某员工行 → 查看公开荣誉摘要（非隐私字段）

---

**角色：** 主管  

**目的：** 查看本团队协作关系（与管理视角）  

**使用路径：**

1. 登录 → `/me/partners`（主管也可进入个人中心）
2. 默认「拍档列表」：共获次数 Top N
3. 切换 Tab「关系网络」→ 深色金色星座图
4. 点击节点 / 连线 → 查看共获明细

## 3. 普通员工

---

**角色：** 员工  

**目的：** 一眼看到自己的荣誉总量与近期成就  

**使用路径：**

1. 登录 → 进入 `/me`（个人中心）
2. 首屏：大号金色数字展示累计荣誉积分、获激励次数、收到点赞数
3. 向下滚动「荣誉明细」时间轴：每条含标题、发放人、时间、分值
4. 点击「查看全部」→ `/me/honors`

---

**角色：** 员工  

**目的：** 了解谁在团队中最突出、自己所处位置  

**使用路径：**

1. 登录 → `/me` 或导航「排行榜」→ `/leaderboard`
2. 个人版 Tab：查看本团队 / 全公司排名
3. 底栏固定「我的名次：第 X 名」（未进 Top N 也展示）
4. 切换周期（本周 / 本月 / 累计）查看变化
5. 切换团队版 Tab：看本团队在全公司的位置及团队内成员排名

---

**角色：** 员工  

**目的：** 找到合作最多的同事，感受协作价值  

**使用路径：**

1. 登录 → `/me` → 点击「最佳拍档」→ `/me/partners`
2. 默认展示拍档卡片列表：头像、姓名、共获次数、最近共获时间
3. Top 1 拍档高亮金框标注
4. 切换 Tab「关系网络」→ 以「我」为中心的星座关系图
5. 点击某拍档 → 侧栏滑出「共获荣誉明细」列表
6. 切换 Tab「共获列表」→ 表格/名录式查看所有共获关系

---

**角色：** 员工  

**目的：** 查看主管分配的任务及进度  

**使用路径：**

1. 登录 → `/me` → 点击「我的任务令」→ `/me/tasks`
2. 查看任务列表：进行中 / 已完成 / 已过期（outline 标签区分）
3. 点击某任务 → 展开详情：描述、截止时间、关联荣誉（若有）
4. （二期）点击「提交完成」→ 等待主管确认

---

**角色：** 员工  

**目的：** 了解团队整体荣誉与成员表现  

**使用路径：**

1. 登录 → 导航「团队展播」→ `/team`
2. 顶部：团队荣誉总量、本月新增、人均荣誉
3. 中部：成员卡片网格（金框头像 + 荣誉值 + 近期成就摘要）
4. 下部：团队近期荣誉事件流
5. 点击成员卡片 → 查看公开荣誉；点击心形图标为同事点赞

---

**角色：** 员工  

**目的：** 对同事的成就表示认可  

**使用路径：**

1. 在 `/team` 成员卡片、或 `/broadcast` 展播卡片、或排行榜行上
2. 点击细线心形「点赞」图标
3. 激活态：金色填充 + 轻微光晕；计数 +1
4. 不可给自己点赞；同一目标每日限赞 1 次（重复点击提示「今日已赞」）

---

**角色：** 员工  

**目的：** 感受团队荣誉氛围（工位 / 手机浏览）  

**使用路径：**

1. （无需登录）访问 `/broadcast`
2. 浏览近期荣誉动态列表；或等待自动轮播
3. 点击某条 → 展开详情与获激励人
4. 点击「全屏展播」→ 适合手机横屏或办公室副屏

## 4. 超级管理员（二期 / 小团队可与主管合并）

---

**角色：** 超级管理员  

**目的：** 维护组织架构与用户账号  

**使用路径：**

1. 登录 → `/admin`
2. 进入「用户与组织」：部门树 + 用户列表
3. 新建 / 编辑用户：用户名、姓名、角色、所属部门
4. 禁用 / 启用账号
5. （可选）Excel 批量导入

---

**角色：** 超级管理员  

**目的：** 配置激励类型与系统参数  

**使用路径：**

1. 登录 → `/admin` → 「激励类型配置」
2. 新增类型：名称、默认分值、勋章图标预览
3. 进入「系统设置」：展播轮播间隔、单次发放人数上限等
