# 奢侈品级网页 UI 设计提示词（Master Prompt）

> 本文档为「荣耀之窗」系统的视觉设计母版提示词。  
> 适用于：Figma 设计、AI 出图（Midjourney / DALL·E）、前端开发视觉对齐。  
> 设计理念参考：Hermès、Bottega Veneta、Aman Resorts、Patek Philippe 数字官网——**克制、留白、材质感、仪式感**。

---

## 一、核心设计提示词（可直接复制）

```
设计一套高端奢侈品品牌官网界面，风格对标 Hermès、Bottega Veneta、Aman Resorts 的数字体验。

【整体气质】
- 极简主义（Minimal Luxury），少即是多；拒绝廉价渐变、霓虹色、过度圆角、卡通插画
- 静谧、克制、有仪式感；每一屏都像杂志内页，而非普通 SaaS 后台
- 传递「稀缺、专属、被珍视」的情绪——荣誉是被郑重呈献的，而非廉价贴纸

【色彩】
- 主色：深邃黑 #0A0A0A、墨夜 #121212
- 底色：象牙白 #F7F3ED、暖米 #EDE6DA、羊皮纸 #E8E0D4
- 点缀：香槟金 #C9A962、古铜金 #A68B4B、淡金箔 #D4BC82（仅用于关键荣誉元素，面积 < 8%）
- 辅助：石墨灰 #6B6B6B、雾银 #9A9A9A
- 禁止：高饱和蓝紫、荧光绿、Material Design 默认色板

【字体】
- 标题：优雅衬线体（如 Cormorant Garamond、Playfair Display、思源宋体 SemiBold）
- 正文：现代无衬线（如 Inter、Helvetica Neue、思源黑体 Light）
- 英文品牌名：全大写 + 宽字距 letter-spacing: 0.2em
- 数字（荣誉积分/排名）：等宽或细衬线，大号展示，像表盘读数

【布局与留白】
- 大量留白，内容区最大宽度 1200px，两侧呼吸空间充足
- 12 列栅格，元素对齐严格；卡片间距 24–40px
- 信息层级不超过 3 层；一屏一个视觉焦点
- 采用 editorial layout（编辑式排版）：大标题 + 细线分割 + 少量精炼文案

【材质与质感】
- 背景：极淡噪点纹理（grain）模拟高级纸张；或深色区微磨砂
- 卡片：毛玻璃（backdrop-blur 12–20px）+ 1px 香槟金半透明描边
- 阴影：极柔和弥散阴影，如 0 24px 80px rgba(0,0,0,0.12)，避免硬边投影
- 分割线：0.5px 金色渐变线或浅灰细线
- 图标：极细线稿（1–1.5px stroke），金色或深灰，禁止填充式卡通图标

【动效】
- 缓慢、优雅：过渡 400–600ms，缓动 cubic-bezier(0.4, 0, 0.2, 1)
- 页面进入：淡入 + 轻微上移（8–16px），错峰 80ms
- 荣誉揭晓：金色微光扫过（shimmer）或粒子缓慢飘散，时长 1.2–2s
- 悬停：轻微亮度提升或描边显现，无弹跳、无抖动
- 大屏轮播：交叉淡入淡出（crossfade），每帧停留 8–12 秒

【摄影与图像】
- 人像：柔和侧光、浅景深、中性背景；圆形头像带 1px 金圈
- 空状态：抽象几何或单色线条插画，非表情包风格
- 勋章/徽章：金属浮雕质感，哑光金 + 细微高光

【组件风格】
- 按钮：主按钮深底金字或金边透明底；圆角 2–4px（近乎直角，偏锐利）
- 输入框：底线式（underline only）或极细边框，聚焦时金色底线延展
- 表格/排行榜：无竖线，仅横线分隔；行高充裕（56–72px）；前三名用金/银/铜细线左边框
- 标签：outline 风格，小字号全大写，字距宽
- 导航：顶部极简，Logo 左，文字链右，无厚重底栏

【禁忌清单】
- 不用 Bootstrap 默认蓝、Ant Design 默认圆角卡片堆叠感
- 不用 emoji 作为核心图标（🏆 仅作辅助，主视觉用定制 SVG）
- 不用密集数据表格压迫感；排行榜像「名录」而非 Excel
- 不用廉价 loading 转圈；用金色细线描边动画或呼吸光点

【情感关键词】
静谧 · 尊贵 · 克制 · 仪式感 · 专属 · 永恒 · 被看见 · 值得铭记
```

## 二、分场景衍生提示词

### 2.1 登录页

```
奢侈品官网登录页。全屏深色背景 #0A0A0A，中央偏上品牌字标「荣耀之窗 WINDOW OF HONOR」衬线大写金字。
底部象牙色登录卡片，毛玻璃质感，仅用户名/密码底线输入框，一枚金色细线边框按钮「进入」。
背景极淡金色粒子缓慢漂浮。无多余装饰。电影级构图，16:9。
```

### 2.2 荣誉展播大屏

```
高端酒店大堂数字艺术屏。深色底，单条荣誉信息居中展示：员工肖像圆形金框，衬线标题「月度之星」，
香槟金细线分隔，下方灰色小字说明。右下角极简时钟。整体像 Rolex 广告屏，非电视台字幕条。
自动轮播感，大量留白，信息不超过 5 个元素。
```

### 2.3 激励排行榜

```
奢侈品会员名录风格排行榜。象牙纸色背景，每行一位获得者：左侧细衬线名次数字，
中央姓名与头衔，右侧荣誉积分用金色大号数字。前三名行背景极淡金箔纹理。
无表格竖线，行间距宽大。顶部筛选器为下划线式 tab。像 Forbes 榜单 meets Hermès 画册。
```

### 2.4 关系网络（最佳拍档）

```
深色背景上的金色星座关系图。节点为圆形人像金框，连线为极细香槟金光线，线粗细表示共获次数。
中心节点略大、微光晕。整体像高端腕表背透机芯或星空图，神秘而精致。
无科技蓝网格底图。缓慢旋转或呼吸动效。
```

### 2.5 主管发放激励

```
高端定制服务下单界面。左侧员工选择列表像贵宾名册，右侧激励表单极简：
称号输入、类型选择为金色 outline 标签。底部「呈递荣誉」金色按钮。
流程庄重，像签署一份荣誉证书，而非填工单。
```

## 三、设计 Token 速查

| Token | 值 | 用途 |
|-------|-----|------|
| `--color-bg-primary` | `#0A0A0A` | 深色主背景、大屏 |
| `--color-bg-secondary` | `#F7F3ED` | 浅色内容区 |
| `--color-accent-gold` | `#C9A962` | 荣誉、按钮、强调 |
| `--color-accent-gold-muted` | `#A68B4B` | 次要金色 |
| `--color-text-primary` | `#1A1A1A` | 浅色区主文字 |
| `--color-text-inverse` | `#F7F3ED` | 深色区主文字 |
| `--color-text-muted` | `#6B6B6B` | 辅助说明 |
| `--font-display` | `Cormorant Garamond, 思源宋体, serif` | 标题 |
| `--font-body` | `Inter, 思源黑体, sans-serif` | 正文 |
| `--radius-sm` | `2px` | 按钮、卡片 |
| `--radius-avatar` | `50%` | 头像 |
| `--shadow-elevated` | `0 24px 80px rgba(0,0,0,0.12)` | 浮起卡片 |
| `--border-gold` | `1px solid rgba(201,169,98,0.4)` | 金边 |
| `--transition-slow` | `500ms cubic-bezier(0.4,0,0.2,1)` | 主过渡 |
| `--spacing-section` | `80–120px` | 区块间距 |
